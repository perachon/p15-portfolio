Rapport de conduite de projet AI Engineering — Extrayez des données multimodales de sites web / pipeline fake news

1. Contexte et objectif

  • Organisation (scénario) : CheckIt.AI, start-up de détection automatique de désinformation.
  • Besoin : pipeline d’acquisition de données multimodales (texte + image) pour entraîner/enrichir un détecteur de fake news.
  • Objectif : automatiser extraction → transformation → stockage avec maintenabilité, reproductibilité et planification.

2. Stratégie d’extraction (extraits)

  • Identifier au moins 3 sources multimodales pertinentes (API, sites web, datasets, flux RSS…).
  • Caractériser chaque source : type de données, format (CSV/JSON/API), langue, qualité des labels (vrai/faux), méthode d’extraction (API REST, Scrapy, Selenium…).
  • Points d’attention : droits d’usage, légalité du scraping, quotas/clefs d’API, association texte‑image, distinction opinions vs désinformation.

3. Scripts d’extraction automatisée (extraits)

  • Scripts Python modulaires, exécutables sans intervention manuelle.
  • Sauvegarde dans une structure cohérente (ex. JSON/CSV ; formats de sortie évoqués : parquet possible).
  • Bonnes pratiques : fonctions (connexion/parsing/nettoyage/sauvegarde), logs, gestion d’erreurs, paramètres configurables.

4. Transformation et schéma de données (extraits)

  • Pipeline reproductible et journalisé : lecture → traitement → export.
  • Transformations : nettoyage, normalisation, mapping, génération de colonnes.
  • Schéma conceptuel (PDF/Mermaid) : champs, types, rôle dans l’usage IA (ex. titre, contenu, URL image, métadonnées).

5. Orchestration ETL avec Apache Airflow (extraits)

  • DAG Airflow orchestrant extraction → transformation → chargement.
  • Tâches distinctes, exécution locale, preuves d’exécution (logs/captures UI).
  • Recommandation : utiliser des PythonOperator simples et réutiliser les fonctions du pipeline.

6. Évaluation et monitoring (extraits)

  • KPI : qualité des données, rapidité, coût.
  • Tableau de bord + plan de monitoring pour la production.
